# 考研线性代数几何图像化学习系统 (Linear Algebra Geometric Visualizer)

这是一个专为研究生入学考试（考研）线性代数复习设计的**高颜值、强交互性的 3D 沉浸式几何学习系统**。

系统以**“用几何直觉映射代数结论”**为核心灵魂，将考研大纲中最抽象、最容易混淆的代数概念（行列式、矩阵形变、方程组交集、张成空间与秩、特征值特征向量、施密特正交化、合同相似与正定二次型）全部转化为直观的 2D/3D 动态模型，并建立了**代数变量 ↔ 几何实体的同色联动高亮映射**。

---

## 🌟 核心功能模块

1. **行列式的几何意义 (`|A|`)**
   * **代数 ↔ 几何映射**：列向量 $\vec{\alpha}_i$ 映射为平行六面体的三条彩色棱边；行列式的值 $|A|$ 映射为六面体的**有向体积 $V$**。
   * **考研映射**：拖拽滑块使其线性相关时，体积坍缩为 0（降维），直观展现 $|A|=0$ 与可逆性、相关性的等价性。

2. **矩阵与线性变换 (`Ax`)**
   * **代数 ↔ 几何映射**：矩阵 $A$ 映射为空间坐标格网的**剪切/拉伸形变器**；输入向量 $\vec{x}$ 与输出 $A\vec{x}$ 映射为形变前后的坐标点。
   * **考研映射**：一键运行复合变换动画（空间连续形变），几何化理解结合律；演示逆矩阵 $A^{-1}$ 恢复扭曲格网。

3. **方程组解的几何空间 (`x=b`)**
   * **代数 ↔ 几何映射**：单条三元一次方程映射为 3D 空间中的**半透明彩色平面**；方程组的解集映射为**平面的公共交集**。
   * **考研映射**：直观演示“无解（形成三棱柱，交线平行无交点）”、“唯一解（交于单点）”和“无穷多解（交于一线或面，展示自由变量与基础解系）”。

4. **向量相关性与秩 (`Span`)**
   * **代数 ↔ 几何映射**：一组向量映射为 3D 空间有向箭头；**张成空间 (Span)** 映射为它们覆盖的几何外壳（1D线、2D面、3D体）；**秩 (Rank)** 映射为**张成空间的几何维度值**。
   * **考研映射**：高亮极大线性无关组，拖拽测试向量落入已有平面即发生相关性降维。

5. **特征值与特征向量 (`λv`)**
   * **代数 ↔ 几何映射**：特征向量方向映射为空间拉伸中**“方向完全不动”的特殊特征直线**；特征值 $\lambda$ 映射为该方向的**长度缩放倍数**。
   * **考研映射**：磁吸游戏设计。用鼠标拖拽测试向量，一旦与特征直线重合，则立刻高亮发光，解锁 $A\vec{x} = \lambda\vec{x}$，几何化阐释特征值真谛。

6. **施密特正交化 (`α⊥β`)**
   * **代数 ↔ 几何映射**：向量点积与除法映射为**正交投影线段**；误差向量映射为天然 90° 的垂直连线。
   * **考研映射**：提供精美的 3D 分步拆解动画。一键拆解从无关向量开始，如何分步剥离投影，拉出相互垂直的正交基，并收缩单位化为刚性坐标直角架。

7. **相似、合同与正定 (`xᵀAx`)**
   * **代数 ↔ 几何映射**：二次型函数 $f(x,y)$ 映射为 3D 弯曲曲面 $z = x^T A x$；特征值映射为主对称轴的**弯曲曲率**；顺序主子式 $D_1, D_2$ 映射为曲面的**拓扑形状控制开关**。
   * **考研映射**：
     * **正定矩阵** $\rightarrow$ **开口朝上的大饭碗**（所有高度 $z > 0$）；
     * **负定矩阵** $\rightarrow$ **倒扣的碗**；
     * **半正定** $\rightarrow$ **长槽谷**；
     * **不定矩阵** $\rightarrow$ **马鞍面**（一头往上弯，一头往下凹）。
     * **合同不变性（惯性定理）**：演示合同剪切拉伸，证明曲面向上弯和向下弯的主轴数守恒，大碗绝不能拉成马鞍。

## 🚀 本地运行指南

由于系统采用纯原生前端技术构建，**无需安装任何繁琐的 Node.js 依赖和打包工具**，解压即可运行！

1. 双击 `index.html` 即可直接在您的默认浏览器中打开。
2. 推荐使用 VS Code 的 **Live Server** 插件，或在命令行通过轻量服务器运行：
   ```bash
   # 进入目录
   cd D:\linear
   # 使用 Python 启动本地临时静态服务器
   python -m http.server 8000
   ```
   然后打开浏览器访问 `http://localhost:8000` 即可流畅体验。

## ☁️ 部署至 GitHub Pages 方案

您可以直接免费挂载到 GitHub Pages，分享给其他考研战友在线复习：

1. 在您的 GitHub 上新建一个公开的空仓库，命名为 `linear-algebra-geometry`。
2. 在您的本地 `D:\linear` 目录打开终端（如 Git Bash），运行以下命令将项目推送到 GitHub：
   ```bash
   git init
   git add .
   git commit -m "feat: 考研线代几何化系统首版初始化"
   git branch -M main
   git remote add origin https://github.com/<您的GitHub用户名>/linear-algebra-geometry.git
   git push -u origin main
   ```
3. 打开 GitHub 仓库页面，点击 **Settings** -> **Pages**：
   * 在 **Build and deployment** 下的 **Branch**，选择 `main` 分支，目录选择 `/ (root)`。
   * 点击 **Save**。
4. 约一分钟后刷新页面，GitHub 就会为您生成专属在线链接：`https://<您的用户名>.github.io/linear-algebra-geometry/`，即可在线无障碍流畅加载！
