# 考研线性代数几何图像化学习系统 (v2.0 全大纲覆盖版)

这是一个专为全国硕士研究生入学考试（考研）线性代数科目打造的**高颜值、强交互、全考点覆盖的 3D 沉浸式几何学习平台**。

系统突破性地引入了**“双画布三维形变对比”**、**“仿射零空间平行平移”**、**“特征主轴正交刚性旋转”**等前沿数学可视化，完美揭示了考研线性代数中最隐蔽、最深奥的代数联系。

## 🌟 六大学术章节与十六大核心考点

系统依照标准考研线代复习脉络，划分为 **6 大章节与 16 个交互模块**：

### 📂 第一章：行列式与矩阵基础
*   **行列式的几何有向体积**：3D 平行六面体有向体积 $V$。拖拽滑块控制高度降为 0，展示 $|A|=0$ 触发“维度坍缩”与线性相关的几何本质。
*   **矩阵转置对称镜像**：**【双画布对比】** 展示转置矩阵 $A^T$ 与 $A$ 关于对角反射面 $y=x$ 的几何对称反射性。
*   **逆矩阵与伴随矩阵**：**【双画布对比】** 左侧展示逆矩阵 $A^{-1}$ 将空间还原的倒退动画；右侧展示伴随矩阵 $A^*$ 携带行列式因子进行 $|A|$ 倍缩放的物理图像。

### 📂 第二章：矩阵之间的四大核心关系
*   **等价关系 ($PAQ$)**：**【双画布对比】** 变换前后的输入输出基底双重更换。证明其不改变值域空间的物理维度（秩守恒：$r(A)=r(B)$）。
*   **相似关系 ($P^{-1}AP$)**：**【双画布对比】** 左侧展示标准基下的剪切形变网格，右侧展示相似特征基下的纯轴向等比拉伸网格。直观领会“特征值是空间形变的本质拉伸基因”。
*   **合同关系 ($C^TAC$)**：**【双画布对比】** 底面网格的剪切扭曲，直观展现二次曲面虽被拉扯，但向上/下弯曲的特征主轴方向数量绝对守恒（惯性定理）。
*   **正交矩阵 ($Q^TQ=I$)**：**【双画布对比】** 正交变换作为“绝对刚性空间旋转器”，夹角和长度完全不变。

### 📂 第三章：向量空间与相关性基础
*   **张成空间与秩**：一组向量能铺满的几何平面/体，其极大无关组为最简骨架。**秩即为张成空间的几何维度值**。
*   **基底与坐标变换**：**【2D拖拽交互】** 同一个向量在新基偏斜坐标网格与标准直角网格下的刻度值动态转换。
*   **向量组间的等价**：**【双画布对比】** 两组方向长短完全不同的向量组，在 3D 中投射出完全重合的同一个 Span 子空间平面。
*   **施密特正交化**：以 3D 渐进式投影剥离动画，直观拆解剥离投影误差、拉直为 90° 直角刚性坐标架的过程。

### 📂 第四章：线性方程组解空间体系
*   **齐次解空间 (核/零空间)**：穿过原点的翠绿色 1D 解线（基础解系方向向量的全部组合 $k\vec{\xi}_1$）。
*   **非齐次解空间 (平移仿射)**：**【双画布对比】** 右侧非齐次解线 $L_b$ 被架空，但与左侧齐次解线 $L_0$ 严格平行。通过亮白特解向量 $\vec{x}_0$ 动态映射 $\vec{x} = \vec{x}_0 + k\vec{\xi}_1$ 的平移几何美感。

### 📂 第五章：特征值与对角化
*   **特征向量磁吸游戏**：**【2D磁吸交互】** 鼠标任意拽动测试向量，一旦贴近特征不动直线即发生高亮“吸附”，并亮起绿色霓虹灯。
*   **对角化与正交对角化**：**【双画布对比】** 实对称矩阵正交对角化 $Q^T A Q = \Lambda$ 的本质是**利用正交矩阵（刚性旋转 45°）将斜摆的特征向量轴刚性旋转对齐到标准直角坐标轴上**。

### 📂 第六章：二次型与正定性深度剖析
*   **二次曲面与正定判定**：实对称矩阵定义二次曲面 $z = x^TAx$。依据 Sylvester 顺序主子式与特征值，曲面即时在“朝上圆碗（正定）”、“倒扣碗（负定）”、“槽谷（半正定）”、“马鞍面（不定）”间进行拓扑质变。
*   **标准化与规范化**：**【双画布对比】** 标准化（左）通过旋转主轴消去交叉项，形成椭圆碗；规范化（右）进一步伸缩坐标轴，**将任意椭圆抛物面碗规范规整为标准的正圆碗**（系数为 1, -1, 0）。
*   **合同变换与惯性定理**：底面网格拉伸下碗拓扑类型严格守恒。

## 🚀 极简免安装 GitHub Pages 一键部署工具

新版部署工具**不依赖您的电脑是否安装 Git，甚至不需要您输入用户名和仓库名！**

1.  直接在 **`D:\linear`** 目录下双击运行 **`deploy.bat`**。

4.  一键完成后，直接点击终端屏幕上印出的绿色高亮专属链接，即可全球在线复习！
